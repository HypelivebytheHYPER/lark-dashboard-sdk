## üéØ Quick Install Commands

### For New Projects
```bash
# Create new project
mkdir my-lark-dashboard
cd my-lark-dashboard
npm init -y

# Install SDK
npm install @hypelab/lark-dashboard-sdk

# Install TypeScript (optional)
npm install -D typescript @types/node

# Create config
echo 'LARK_API_KEY=your-key-here' > .env
```

### For Existing Projects
```bash
# Install SDK
npm install @hypelab/lark-dashboard-sdk

# Update package.json scripts
npm pkg set scripts.dashboard="ts-node dashboard.ts"
```

### For MCP Usage
```bash
# Install globally for MCP
npm install -g @hypelab/lark-dashboard-sdk

# Or use with npx
npx @hypelab/lark-dashboard-sdk
```
